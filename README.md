# Development
## Installation requirements
* Node
* Yarn

## Build Setup

Install dependencies:
```
$ yarn install
```

Run server:
```
$ yarn start
```

Serve frontend with hot reload:
```
$ yarn dev
```

Build frontend:
```
$ yarn build
```
